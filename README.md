## Foundations & References

- [Data Science Stack Exchange](https://datascience.stackexchange.com/) – Community Q&A covering theory, tools, and practice.
- [arXiv Data Science](https://arxiv.org/) – Open-access preprints across statistics, ML, and data analysis.
- [CRISP-DM](https://www.ibm.com/topics/crisp-dm) – Widely used process model for data mining projects.
- [KDnuggets](https://www.kdnuggets.com/) – News, tutorials, and opinions in data science and analytics.
- [Towards Data Science](https://towardsdatascience.com/) – Popular publication with practical data science articles.

## Programming Languages

- [Python](https://www.python.org/) – Dominant language for data science, ML, and scientific computing.
- [R](https://www.r-project.org/) – Statistical computing language with rich analysis packages.
- [Julia](https://julialang.org/) – High-performance language for numerical and scientific computing.
- [Scala](https://www.scala-lang.org/) – JVM language commonly used with Apache Spark.
- [SQL](https://en.wikipedia.org/wiki/SQL) – Query language essential for data extraction and analysis.

## Data Wrangling & ETL

- [pandas](https://pandas.pydata.org/) – Core Python library for data manipulation and analysis.
- [Polars](https://www.pola.rs/) – Fast DataFrame library optimized for performance.
- [Apache Airflow](https://airflow.apache.org/) – Workflow orchestration platform for data pipelines.
- [dbt](https://www.getdbt.com/) – Transformation tool for analytics engineering workflows.
- [Apache NiFi](https://nifi.apache.org/) – Visual tool for automating data flows between systems.
- [Talend Open Studio](https://www.talend.com/products/talend-open-studio/) – Open-source ETL and integration platform.

## Exploratory Data Analysis

- [ydata-profiling](https://github.com/ydataai/ydata-profiling) – Automated EDA reports for pandas DataFrames.
- [Sweetviz](https://github.com/fbdesignpro/sweetviz) – Visual EDA tool for quick dataset inspection.
- [PandasGUI](https://github.com/adamerose/PandasGUI) – Interactive GUI for exploring pandas DataFrames.
- [D-Tale](https://github.com/man-group/dtale) – Interactive visualizer for pandas and NumPy data.

## Visualization

- [Matplotlib](https://matplotlib.org/) – Foundational plotting library for Python.
- [Seaborn](https://seaborn.pydata.org/) – Statistical data visualization built on Matplotlib.
- [Plotly](https://plotly.com/python/) – Interactive plotting library for dashboards and notebooks.
- [Altair](https://altair-viz.github.io/) – Declarative statistical visualization library for Python.
- [Tableau](https://www.tableau.com/) – Business intelligence and data visualization platform.
- [Power BI](https://powerbi.microsoft.com/) – Analytics and visualization service by Microsoft.

## Statistics & Probability

- [SciPy Stats](https://docs.scipy.org/doc/scipy/reference/stats.html) – Statistical functions for scientific computing.
- [Statsmodels](https://www.statsmodels.org/) – Statistical modeling and hypothesis testing in Python.
- [Stan](https://mc-stan.org/) – Probabilistic programming for Bayesian inference.
- [PyMC](https://www.pymc.io/) – Bayesian statistical modeling and probabilistic ML.
- [scikit-posthocs](https://github.com/maximtrp/scikit-posthocs) – Post-hoc statistical tests for Python.

## Machine Learning

- [scikit-learn](https://scikit-learn.org/) – Core ML library for classical algorithms in Python.
- [XGBoost](https://xgboost.ai/) – Gradient boosting library for structured data.
- [LightGBM](https://lightgbm.readthedocs.io/) – Fast gradient boosting framework by Microsoft.
- [CatBoost](https://catboost.ai/) – Gradient boosting with strong categorical feature support.
- [MLflow](https://mlflow.org/) – Platform for tracking experiments and managing ML lifecycles.

## Deep Learning

- [TensorFlow](https://www.tensorflow.org/) – End-to-end deep learning framework.
- [PyTorch](https://pytorch.org/) – Popular deep learning library with dynamic computation graphs.
- [Keras](https://keras.io/) – High-level neural networks API.
- [Hugging Face Transformers](https://huggingface.co/transformers/) – Pretrained models for NLP and multimodal tasks.
- [FastAI](https://www.fast.ai/) – High-level library simplifying deep learning workflows.

## Time Series & Forecasting

- [statsforecast](https://github.com/Nixtla/statsforecast) – Fast statistical forecasting library.
- [Prophet](https://facebook.github.io/prophet/) – Time series forecasting tool for business use cases.
- [GluonTS](https://github.com/awslabs/gluonts) – Probabilistic time series modeling by AWS.
- [Darts](https://github.com/unit8co/darts) – Python library for easy time series forecasting.
- [tslearn](https://tslearn.readthedocs.io/) – Machine learning toolkit for time series data.

## Big Data & Distributed Computing

- [Apache Spark](https://spark.apache.org/) – Distributed data processing engine.
- [Apache Hadoop](https://hadoop.apache.org/) – Framework for distributed storage and processing.
- [Dask](https://www.dask.org/) – Parallel computing library that scales Python workflows.
- [Ray](https://www.ray.io/) – Distributed computing framework for ML and Python apps.
- [Flink](https://flink.apache.org/) – Stream and batch processing framework.

## Databases & Storage

- [PostgreSQL](https://www.postgresql.org/) – Advanced open-source relational database.
- [MySQL](https://www.mysql.com/) – Popular relational database system.
- [MongoDB](https://www.mongodb.com/) – NoSQL document-oriented database.
- [DuckDB](https://duckdb.org/) – In-process analytical SQL database.
- [BigQuery](https://cloud.google.com/bigquery) – Serverless data warehouse on GCP.
- [Snowflake](https://www.snowflake.com/) – Cloud-native data warehouse platform.

## MLOps & Production

- [Kubeflow](https://www.kubeflow.org/) – Kubernetes-native ML workflows.
- [Seldon Core](https://www.seldon.io/) – Model deployment and inference on Kubernetes.
- [BentoML](https://bentoml.com/) – Framework for serving ML models in production.
- [Weights & Biases](https://wandb.ai/) – Experiment tracking and model monitoring.
- [Evidently](https://www.evidentlyai.com/) – Data and model monitoring for ML systems.

## Notebooks & Experimentation

- [Jupyter](https://jupyter.org/) – Interactive notebooks for data analysis and ML.
- [Google Colab](https://colab.research.google.com/) – Cloud-hosted Jupyter notebooks with free GPUs.
- [Kaggle Notebooks](https://www.kaggle.com/notebooks) – Collaborative notebooks with datasets and competitions.
- [VS Code Notebooks](https://code.visualstudio.com/docs/datascience/jupyter-notebooks) – Notebook support inside VS Code.

## Datasets & Open Data

- [Kaggle Datasets](https://www.kaggle.com/datasets) – Public datasets for data science projects.
- [UCI ML Repository](https://archive.ics.uci.edu/ml/) – Classic datasets for ML research.
- [OpenML](https://www.openml.org/) – Open datasets and benchmarks for ML experiments.
- [Google Dataset Search](https://datasetsearch.research.google.com/) – Search engine for public datasets.
- [World Bank Open Data](https://data.worldbank.org/) – Global development and economic data.
